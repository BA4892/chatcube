【新增】
1. 新增 OFD 文档解析支持，更多文档附件可直接识别和读取。
2. 新增 OpenAI Responses 生图主链路支持，图片生成流程更完整。
3. 新增支持图片预览时的系统分享。
4. 新增两款主题配色。

【优化】
1. 优化聊天流式回复体验，内容输出更自然、更稳定。
2. 优化聊天滚动、消息地图跳转、展开内容定位和回到底部体验。
3. 优化图片保存，现在只会弹一次确认弹窗。
4. 优化主题配色、聊天标题栏、设置弹层等界面细节，更贴近 HarmonyOS 原生体验。
5. 优化联网搜索、Responses 格式适配和模型兼容表现。
6. 完善数据备份与导入导出能力，提升生成图片和服务商图标等数据的保留完整性。

【修复】
1. 修复连续发送图片附件时可能出现串图的问题。
2. 修复长截图分享时可能出现颜色异常的问题。
3. 修复联网搜索偶发失败、搜索结果展示不稳定的问题。
4. 修复生图结果顺序异常、结果保存不完整等问题。
